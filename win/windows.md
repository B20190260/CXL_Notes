## tree 生成目录树
win2K以上的操作系统中都带有tree命令，98下面没有，就算是把2K下面的拷过去也用不起来。  
tree命令的格式是tree [drive][path] [/F] [/A]  
/F 显示每个文件夹中文件的名称  
/A 使用ASCII字符，而不使用扩展字符  
可以在命令行窗口敲"tree /?"看帮助。  
格式：TREE【盘符：】【F】【>PRN】  
使用/F参数时显示所有目录及目录下的所有文件，省略时，只显示目录，不显示目录下的文件；
选用>PRN参数时，则把所列目录及目录中文件名打印输出  

例如：  
```
  tree /f >tree.txt
```
